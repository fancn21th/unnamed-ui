Message 组件提供了用于显示 AI 消息和用户消息的基础样式组件，适用于聊天界面、对话系统等场景。

## Features

- **两种消息类型**：`AIMessage`（AI 消息，无背景色）和 `UserMessage`（用户消息，带背景色）
- **状态管理**：AI 消息支持三种状态（正常、生成中、生成失败），方便展示消息生成过程
- **反馈区域**：消息下方预留反馈区域，方便添加点赞、点踩、复制等操作按钮
- **样式隔离**：消息样式不受 fumadoc 等文档框架影响
- **灵活扩展**：支持自定义内容和样式
