# CandidatesTable 候选人信息表格

竖向表格展示候选人详细信息，第一列为字段名（表头），后续列为各候选人的对应值，支持多个候选人横向对比。

## 特性

- ✅ 竖向表格布局，便于对比
- ✅ 支持多个候选人横向对比
- ✅ 展示姓名、年龄、工作年限、学历等关键信息
