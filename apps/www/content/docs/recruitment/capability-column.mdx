# CapabilityColumn 能力对比柱状图

展示候选人能力维度对比分析的柱状图列表组件，通过多个柱状图直观对比候选人在不同能力维度上的表现。

## 特性

- ✅ 多个能力维度柱状图展示
- ✅ 直观的候选人能力对比
- ✅ 两层组件结构：单个柱状图 + 柱状图列表
