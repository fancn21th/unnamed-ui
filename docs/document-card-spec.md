# 文档卡片 (Document Card) 组件设计说明

> 版本：v1.0
> 最后更新：2026-02-09

---

## 1. 组件概述

### 1.1 基本信息
- **组件名称**：文档卡片 (Document Card)
- **用途**：展示研究类文档的基本信息和属性
- **交互类型**：可点击/可选择

### 1.2 视觉基础
| 属性 | 值 |
|------|-----|
| 形状 | 矩形 |
| 圆角 | 12px - 16px |
| 背景色 | 白色 (#FFFFFF) |
| 边框色 | 浅蓝灰色（细边框） |
| 阴影 | 轻微阴影（待确认） |

---

## 2. 布局结构

### 2.1 整体布局
```
┌─────────────────────────────────────┐
│          头部信息区 (Header)          │
├─────────────────────────────────────┤
│           分割线 (Separator)          │
├─────────────────────────────────────┤
│          详细属性区 (Body)            │
└─────────────────────────────────────┘
```

采用 **上下纵向布局**，分为三个主要区域：
1. 头部信息区（顶部）
2. 分割线（中部）
3. 详细属性区（底部）

---

## 3. 头部信息区 (Header)

### 3.1 布局结构
```
┌─────────┬──────────────────────────────────────┐
│  图标          标题（AI趋势研究）               │
│                                                │
│ 更新时间：18:21                                 │
└─────────┴──────────────────────────────────────┘
```

### 3.2 图标区域
| 属性 | 值 |
|------|-----|
| 图标描述 | 线性图标，结合"放大镜/显微镜"与"节点图"意象 |
| 图标颜色 | 黑色 (#000000) |
| 位置 | 卡片左侧，与标题垂直居中对齐 |
| 尺寸 | 待确认（建议 24x24 或 32x32） |

### 3.3 标题 (Title)
| 属性 | 值 |
|------|-----|
| 文本内容 | **AI趋势研究** |
| 字体权重 | 加粗 (Bold) |
| 字号 | 较大（待确认，建议 16px-18px） |
| 颜色 | 深黑色 (#222 或 #333) |
| 对齐方式 | 左对齐 |

### 3.4 更新时间 (Update Time)
| 属性 | 值 |
|------|-----|
| 文本内容 | **更新时间：18:21** |
| 颜色 | 中灰色 (#666 或 #999) |
| 字号 | 较小（建议 12px-14px） |
| 对齐方式 | 左对齐（与标题左侧对齐） |
| 位置 | 标题下方 |

---

## 4. 分割线 (Separator)

| 属性 | 值 |
|------|-----|
| 类型 | 水平分割线 |
| 颜色 | 浅灰色 |
| 宽度 | 贯穿卡片左右 |
| 位置 | 头部信息区与属性区之间 |

---

## 5. 详细属性区 (Body)

### 5.1 布局结构
采用 **垂直列表布局**，包含三行属性信息。

每行结构：
```
标签：┌─────────────────────┐
      │      数值内容        │
      └─────────────────────┘
```

### 5.2 属性列表

#### 属性 1：研究主题 (Research Topic)
```
研究主题：┌──────────┐
         │   标签   │
         └──────────┘
```

| 属性 | 标签部分 | 数值部分 |
|------|---------|---------|
| 标签文本 | `研究主题：` | - |
| 数值内容 | - | `标签` |
| 标签颜色 | 中灰色 (#666/#999) | - |
| 数值颜色 | - | 深色 (#222/#333) |
| 数值容器 | - | 带浅灰色圆角边框、白色背景 |
| 容器圆角 | - | 与卡片圆角一致 |

#### 属性 2：研究领域 (Research Field)
```
研究领域：商业投资
```

| 属性 | 值 |
|------|-----|
| 标签文本 | `研究领域：` |
| 数值内容 | `商业投资` |
| 标签颜色 | 中灰色 (#666/#999) |
| 数值颜色 | 深色 (#222/#333)，普通字体 |
| 数值样式 | 无边框，普通文本 |

#### 属性 3：研究周期 (Research Period)
```
研究周期：2015-2025
```

| 属性 | 值 |
|------|-----|
| 标签文本 | `研究周期：` |
| 数值内容 | `2015-2025` |
| 标签颜色 | 中灰色 (#666/#999) |
| 数值颜色 | 深色 (#222/#333)，普通字体 |
| 数值样式 | 无边框，普通文本 |

## 6. 视觉样式规范

### 6.1 颜色层级
| 用途 | 颜色值 | 示例 |
|------|--------|------|
| 标题、属性值（主要信息） | #222 或 #333 | AI趋势研究、商业投资 |
| 标签、更新时间（辅助信息） | #666 或 #999 | 研究领域：、更新时间：18:21 |
| 边框 | 浅蓝灰色 | 卡片边框 |
| 分割线 | 浅灰色 | 头部与属性区分隔线 |

### 6.2 间距规范
| 区域 | 间距类型 | 建议值 |
|------|---------|-------|
| 卡片内部 | 内边距 (Padding) | 待确认（建议 16px-20px） |
| 属性行之间 | 垂直间距 | 固定间距（建议 8px-12px） |
| 标签与数值之间 | 水平间距 | 紧凑间距（建议 4px-8px） |

### 6.3 对齐规范
| 元素 | 对齐方式 |
|------|---------|
| 所有文本和图标 | 左对齐 |
| 标题与图标 | 垂直居中对齐 |
| 属性标签与数值 | 基线对齐 |

## 11. 组件文件结构

组件包含以下文件：

```
📁 apps/www/registry/wuhan/
├── 📁 blocks/document-card/
│   └── 📄 document-card-01.tsx        # 基础原语组件
├── 📁 composed/document-card/
│   └── 📄 document-card.tsx           # 组合组件
├── 📁 examples/document-card/
│   └── 📄 document-card-demo.tsx     # 示例组件
└── apps/www/content/docs/blocks/
    └── 📄 document-card.mdx          # 组件文档
```

### 11.1 Block 层 (document-card-01.tsx)

提供基础的原语组件，包括：

- `DocumentCardContainerPrimitive` - 卡片容器原语
- `DocumentCardHeaderPrimitive` - 头部信息区原语
- `DocumentCardSeparatorPrimitive` - 分割线原语
- `DocumentCardFieldPrimitive` - 属性行原语
- `DocumentCardDefaultIcon` - 默认文档图标
- `DocumentCardPrimitive` - 完整的卡片原语

### 11.2 Composed 层 (document-card.tsx)

提供可直接使用的组合组件，包括：

- `DocumentCard` - 单个文档卡片组件
- `DocumentCardList` - 文档卡片列表组件

### 11.3 Example 层 (document-card-demo.tsx)

提供组件的使用示例和演示。
