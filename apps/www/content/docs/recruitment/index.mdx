# Recruitment Components

This section contains a comprehensive collection of recruitment-related UI components designed for building HR and talent acquisition applications.

## Overview

The recruitment components library includes various types of components:

### Core Components

- **Job Management**: JobCard, SimilarJob, RecruitmentJobTable
- **Candidate Management**: CandidateReport, CandidatesTable, PersonnelRecommend
- **Interview Process**: InterviewInfoForm, InterviewProcessTable, InterviewQuestionPanel
- **Reports & Analytics**: ReportCard, ReportSection, ResumeEvaluationReport, RecruitmentActivityOverview

### UI Components

- **Forms**: ConfirmJDForm, InterviewInfoForm
- **Lists**: DataSourceList, TodoList, WorkObjectiveList, WorkResultList
- **Modals**: DeleteConfirmModal, RepeatFileModal
- **Input**: ChatRadio, Sender
- **Upload**: ResourceUpload
- **Status**: StatusTag, Thinking
- **Navigation**: StepDetail, StepOverview
- **Charts**: CapabilityColumn
- **Communication**: MessageBubble

### Utility Components

- **Dropdowns**: ActionDropdown
- **Guides**: DatasourceGuide
- **Scenes**: RecruitmentScene

## Installation

All components can be installed individually using the CLI or manually copied into your project. Check each component's documentation page for specific installation instructions.

## Features

- ✅ Built with Ant Design and styled-components
- ✅ Fully typed with TypeScript
- ✅ Responsive and accessible
- ✅ Customizable and extensible
- ✅ Production-ready
