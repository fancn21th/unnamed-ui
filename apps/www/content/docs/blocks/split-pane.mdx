分隔面板，支持左右或上下分割，可调整大小，支持折叠功能。

## 快速开始

```tsx
import { TripleSplitPane } from "@/registry/wuhan/composed/triple-split-pane";

export function Example() {
  return (
    <TripleSplitPane
      left={{
        title: "左侧面板",
        children: <div>左侧内容</div>,
      }}
      center={{
        title: "中间面板",
        children: <div>中间内容</div>,
      }}
      right={{
        title: "右侧面板",
        children: <div>右侧内容</div>,
      }}
    />
  );
}
```

## API

### TripleSplitPane

三栏分隔面板，支持左右面板折叠功能。

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | 容器自定义类名 |
| `left` | `PanelConfig` | - | 左侧面板配置 |
| `center` | `PanelConfig` | - | 中间面板配置 |
| `right` | `PanelConfig` | - | 右侧面板配置 |

### PanelConfig

面板配置对象

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | 面板内容 |
| `title` | `ReactNode` | - | 面板标题 |
| `defaultSize` | `number` | `20`(左)/`50`(中)/`30`(右) | 默认宽度（百分比）|
| `minSize` | `number` | `15`(左/右)/`30`(中) | 最小宽度（百分比）|
| `collapsedSize` | `number` | `0` | 折叠后宽度（像素），设为 0 表示完全折叠 |
| `collapsibleIcon` | `ReactNode` | `<PanelLeft />`(左)/`<PanelRight />`(右) | 折叠图标 |

### SplitPaneItem

带标题和折叠图标的面板项原语组件，基于 react-resizable-panels 构建。

#### Props

继承 react-resizable-panels 的 `PanelProps`，额外包含：

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `panelTitle` | `ReactNode` | - | 面板标题 |
| `collapsibleIcon` | `ReactNode` | `<PanelLeft />` | 折叠图标 |
| `showCollapsibleIcon` | `boolean` | `true` | 是否显示折叠图标 |
| `onCollapsibleClick` | `() => void` | - | 折叠图标点击事件 |
| `headerClassName` | `string` | - | header 的自定义类名 |
| `bodyClassName` | `string` | - | body 的自定义类名 |
| `containerClassName` | `string` | - | 容器的自定义类名 |
| `children` | `ReactNode` | - | 内容区域的子元素 |

## 原语组件

TripleSplitPane 组件基于以下原语组件构建：

- `SplitPaneItem`: 面板项组件，包含标题、折叠按钮和内容区域
- `SplitHandle`: 分隔手柄组件
- `SplitPanelGroup`: 面板组容器

如需更灵活的定制，可以直接使用 `split-pane-01` 中的这些原语组件。
