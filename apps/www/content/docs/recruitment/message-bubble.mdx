# MessageBubble 消息气泡

使用 Antd X 的消息气泡组件，支持打字机效果和操作按钮。AI消息无背景色，用户消息背景色为 #E8EAFF。

## 特性

- ✅ 支持打字机效果
- ✅ 支持操作按钮
- ✅ AI 消息和用户消息不同样式
