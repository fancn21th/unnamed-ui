## 介绍

Markdown 是一个支持流式渲染的 Markdown 组件，可以实时展示 Markdown 内容，支持代码高亮、表格、列表等常见 Markdown 语法。

<ComponentPreview name="markdown-demo" description="Markdown 基础示例" />

## 特性

- 支持流式渲染
- 代码高亮
- 表格渲染
- 自定义组件扩展
- 性能优化（使用 useDeferredValue）

## API

### Markdown Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| content | string | - | Markdown 内容 |
| status | string | - | 消息状态 |
| messageId | string | - | 消息 ID |
